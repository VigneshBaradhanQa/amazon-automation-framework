# amazon-automation-framework

Selenium Test Automation Framework using Java, TestNG, and POM design pattern with reusable components and scalable structure.



\# Amazon Automation Framework



\## Tech Stack

\- Java

\- Selenium WebDriver

\- TestNG

\- Maven



\## Framework Design

\- Page Object Model (POM)

\- Reusable utilities

\- Base test setup



\## Features

\- Login Test

\- Product Search Test

\- Screenshot on failure

\- Extent Reports



\## Project Structure

src/main/java

&#x20; - base

&#x20; - pages

&#x20; - utils



src/test/java

&#x20; - tests



\## How to Run

1\. Clone repo

2\. Run: mvn clean test



\## Reports

\- TestNG Reports → test-output/

\- Extent Reports → reports/
